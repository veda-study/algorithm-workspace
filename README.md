## 💡 스터디 그라운드 룰

> 💡 *지속 가능한 스터디를 위해 서로 약속한 최소한의 규칙입니다.*

* **문제 풀이 및 기록:** 매주 정해진 필수 문제를 풀고, 매일 진척 상황(미완성 포함)을 **Jira 이슈 댓글**에 데일리 로그로 남깁니다.
* **코드 리뷰 필수:** 본인의 코드를 `main`에 직접 Push할 수 없으며, 최소 1명 이상의 **Approve**를 받아야 Merge할 수 있습니다.
* **AI 사용 금지 및 공식 레퍼런스 활용:** 문제 풀이 시 AI에게 코드를 직접 요구하는 행위를 금지합니다. 대신 [C++ 공식 레퍼런스(cppreference.com)](https://ko.cppreference.com/)를 참고하여 문서를 직접 읽고 코드를 분석하는 연습을 지향합니다.
